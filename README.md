# My TabProject 
  > 몰입캠프 1주차 과제
  - Android Studio + Kotlin 사용해보기
  - 3개의 탭 구조를 갖는 안드로이드 앱 개발하기

## 개발 팀원
- 최수연 (숙명여대 컴퓨터과학전공 19)
- 윤인규 (카이스트 전산학부 22)

## 개발 환경
- Language: Kotlin
- IDE: Android Studio

## 화면 소개

####  1번째 탭
- 휴대폰 안에 연락처를 받아와서 전화번호부를 띄워줌 (사진은 연락처를 직접 추가한 상태)
- 전화 걸기/문자 보내기/전화번호부 수정 기능 구현
